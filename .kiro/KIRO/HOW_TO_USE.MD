## ✅ Overview

Use Kiro AI agent FULLY in terminal (no IDE, no VS Code UI).

---

## ⚙️ Run Kiro CLI

```

& "$env:LOCALAPPDATA\Kiro-Cli\kiro-cli.exe"

```

---

## ⚙️ Open Project

```

cd "your-project-path"

```

